# Avi Converter

A simple, lightweight web-based AVI video converter. Just open `index.html` in your browser and start converting!

## 🚀 Quick Start

1. **Download or clone the repository**
   ```bash
   git clone https://github.com/JustWazy/Avi-Converter.git
   ```

2. **Open in your browser**
   - Simply double-click `index.html` or
   - Right-click `index.html` → Open with → Your preferred browser

That's it! No installation, no server setup required.

## 📋 Features

- **Simple & Fast** - No complex setup or dependencies
- **Works Offline** - Runs entirely in your browser
- **Video Conversion** - Convert AVI and other video formats
- **Lightweight** - Minimal file size, quick loading
